input 绑定Dom value 和 html属性
HTML 属性 和 Dom 属性的关系：
少量html属性和Dom属性之间有1：1的映射，如id
有些html属性没有对应的Dom属性，如colspan
有些Dom属性没有对应的html属性，如textContent
就算名字相同，html属性和dom属性也不是同一样东西
html的值指定的是初始的值，Dom属性的值指的是当前的值。 html属性的值不可改变，Dom属性的值可以改变
模板绑定是通过Dom属性和事件来工作的，而不是html属性
